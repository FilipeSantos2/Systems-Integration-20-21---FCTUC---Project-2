# Systems-Integration-20/21 (FCTUC) Project 2

## Objectives

Gain familiarity with the development of tiered enterprise applications using the Java Enterprise Edition (Java EE) model. This includes the development of applications based on Enterprise JavaBeans (EJB), the development of Web Services, the use of a Persistence Engine, and the creation of a web front-end. Overall, you should build at least one Enterprise Archive, which can be deployed in an Application Server.
